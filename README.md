# 📊 Regressão Linear Múltipla

Este repositório contém um exemplo de **regressão linear múltipla** utilizando **Python + NumPy** para calcular manualmente os coeficientes da equação. O objetivo é mostrar como o modelo funciona internamente, sem usar bibliotecas de machine learning.

## 📌 O que o código faz
- Carrega um dataset com: quartos, banheiros, área (sqft) e preço.
- Separa variáveis independentes (X) e dependente (Y).
- Adiciona manualmente o **intercepto (beta₀)**.
- Aplica a **Equação Normal** para calcular os betas:
β = (Xᵀ X)⁻¹ Xᵀ Y
<br>
 - Exibe os coeficientes encontrados.
- Realiza uma previsão de preço usando o modelo.

## 🛠️ Tecnologias
- Python  
- NumPy

## 🎯 Finalidade
Demonstrar, de forma direta e prática, como funciona o cálculo matemático da regressão múltipla e como aplicar o modelo para prever valores.
